# School-Website
Projekt shkollor: website i thjeshtë i ndërtuar me HTML, CSS dhe JavaScript.


# Website i Shkollës

Ky projekt përfaqëson një website informues për shkollë, i realizuar si pjesë e një projekti shkollor. Faqja është ndërtuar për të ofruar informacione të qarta dhe të strukturuara për nxënësit, prindërit dhe stafin arsimor.

## Qëllimi i projektit
Qëllimi kryesor i këtij projekti është aplikimi praktik i njohurive në zhvillimin e faqeve web dhe krijimi i një platforme funksionale dhe të lehtë për përdorim.

## Përmbajtja e faqes
- Informacione rreth shkollës  
- Njoftime dhe aktivitete shkollore  
- Informacione kontakti  

## Teknologjitë e përdorura
- HTML5  
- CSS3  
- JavaScript
